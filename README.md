# Laboratorio DOM - Registro de Estudiantes

Aplicación web para el registro, edición, eliminación y consulta de estudiantes, desarrollada como práctica de manipulación del DOM con JavaScript.

## Tecnologías utilizadas

- HTML5
- CSS3 (Bootstrap 5.3.3)
- JavaScript vanilla
- SweetAlert2 (confirmaciones y alertas visuales)
- Toastify (notificaciones emergentes)

## Funcionalidades

- Registro de estudiantes con nombre, carrera y semestre
- Tabla dinámica con columna de acciones
- Visualización de JSON individual y general mediante modal Bootstrap
- Edición de estudiantes existentes
- Eliminación con confirmación (SweetAlert2)
- Búsqueda por texto (nombre, carrera o semestre)
- Filtro por carrera mediante select
- Contador actualizado en tiempo real
- Limpieza total de registros
- Notificaciones visuales con Toastify

## Cómo ejecutar

1. Abrir el archivo `index.html` en cualquier navegador web moderno.
2. No se requiere servidor local ni instalación de dependencias.

## Archivos del proyecto

- `index.html` - Estructura HTML con formulario, tabla, modal y filtros
- `app.js` - Lógica de JavaScript con manipulación del DOM
- `README.md` - Descripción del proyecto
